## 🆘 IF SOMETHING BREAKS

**Most common issues & fixes:**

| Issue | Fix |
|-------|-----|
| Build fails | Check requirements.txt syntax |
| App won't start | Check logs in Railway dashboard |
| Database error | Verify DATABASE_URL is set in environment |
| 502 error | Wait 30 seconds (containers starting) then refresh |
| Health check fails | Verify SECRET_KEY and JWT_SECRET are set |
| Forgot URL | Check Railway dashboard → Deployments → Settings |
