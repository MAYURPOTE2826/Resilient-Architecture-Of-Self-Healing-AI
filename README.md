# Resilient Architecture for Self-Healing System using Machine Learning

## Project Overview
This project implements an AI-based self-healing architecture capable of detecting anomalies in system metrics and automatically performing recovery actions.

## Features
- Anomaly detection using Isolation Forest
- Fault classification using Random Forest
- Automated recovery (service restart)
- Real-time monitoring
- Grafana dashboard visualization

## Technologies Used
- Python
- Flask
- Machine Learning
- Docker
- Grafana
- Linux
